**Mô tả:**

Dataset gồm 22500 ảnh về đồ vật / rác thải, trong đó có `55.69%` ảnh là vật hữu cơ, `44.31%` ảnh là vật vô cơ

Dataset được chia sẵn tập train / test với tỉ lệ `TEST / TRAIN = 1 / 9`
